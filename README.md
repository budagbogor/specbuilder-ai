# SpecBuilder AI

SpecBuilder AI adalah starter Next.js App Router untuk membuat fondasi aplikasi penyusun spesifikasi proyek.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui style components
- Prisma
- SQLite
- Zod
- OpenAI API
- JSZip

## Menjalankan Project

```bash
npm install
npm run dev
```

Salin `.env.example` ke `.env` jika ingin menyiapkan database lokal.
